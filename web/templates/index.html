IB Integrator

Внутренняя система интегратора по информационной безопасности для ведения справочника клиентов, объектов защиты и проектов по ИБ.

  • Учёт клиентов и контактов по ИБ
  • Объекты защиты: ИСПДн, ГИС, АСУ ТП, корпоративные сети
  • Проекты: аудит, внедрение СЗИ, пентесты, сопровождение
  • Роли пользователей, маскирование контактов, журнал аудита

Для чего система?

Система моделирует работу интегратора по ИБ: позволяет хранить данные о клиентах, их объектах защиты и связанных с ними проектах по информационной безопасности.

В учебных целях демонстрируются: разграничение ролей, проверки корректности ввода, контроль статусов проектов и аудит действий пользователей.
